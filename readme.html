How to start the game ?

For Ajax requests work, you must start the game with a server. Otherwise, you'll get a black screen

  • Window : WAMP
  • Mac OS : MAMP
  • Linux : sudo aptitude install apache2 php5 mysql-server php5-mysql libapache2-mod-php5

Turn on the server, then enter in the address bar of your browser :

http://localhost/[path of my game]/index.html
